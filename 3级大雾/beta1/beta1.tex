拟合曲线为：

\begin{equation}
    \log_{10}{\frac{I}{I_0}} = -0.1213 d + 0.1461
\end{equation}

用前四个点计算得：$h = 0.0504$

用最后4个点计算得:$g(x) = -4.2848 x + 1.5273$

联立后可以解得：$R = \frac{4+1.5273+0.05}{4.2848} = 1.30 g/cm^2$

最终解得：$E = 1.85 \times R + 0.245 = 2.65 MeV$
